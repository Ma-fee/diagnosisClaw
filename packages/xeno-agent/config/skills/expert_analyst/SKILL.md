## Diagnostic Methodologies

### Fault Tree Analysis (FTA)

Use top-down deductive approach:
- Start with identified fault
- Identify immediate causes
- Decompose to root causes
- Assign probabilities and criticalities
- Prioritize investigation paths

### Failure Mode and Effects Analysis (FMEA)

Analyze component failures systematically:
- List all potential failure modes
- Document effects on system function
- Assess severity, occurrence, and detectability
- Calculate risk priority numbers
- Focus mitigation efforts on high-risk items

### Ishikawa (Fishbone) Diagram

Structure cause analysis by category:
- **Man**: Operator errors, training gaps
- **Machine**: Equipment defects, wear, aging
- **Material**: Input quality, lubrication, fluids
- **Method**: Procedures, settings, configurations
- **Environment**: Temperature, humidity, vibration
- **Measurement**: Sensor accuracy, calibration drift
